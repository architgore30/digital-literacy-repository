Working on my digital portfolio was actually a pretty interesting experience. At first, it felt like I was just creating accounts on different platforms, but as I went along, I realized it’s more about how you present yourself online. I created profiles on Kaggle, Stack Overflow, and GitHub, and also made a README repository on my GitHub profile. That README part was something new for me and felt a bit like creating my own mini introduction page.

I also understood that each platform has its own purpose. Kaggle is great for practicing data-related work, Stack Overflow helps when you’re stuck on problems, and GitHub is where you actually show what you’ve built. It made me think more seriously about how others might see my work.

One thing I found a bit difficult was figuring out what to include and how to make it look professional without overdoing it. But at the same time, it was interesting to experiment with how I describe myself and my skills. Overall, this task helped me realize that building a digital presence is important, and it’s something I’ll keep improving over time.
